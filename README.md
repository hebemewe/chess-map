# Chess Map

## Server

Add the following to your MCP server configuration:

"Server": {
  "command": "uvx",
  "args": [
    "--directory",
    "/Users/ac/Desktop/mcp-build-chess/",
    "run",
    "chess"
  ]
},

To run the server locally:

```bash
uvx --directory /Users/ac/Desktop/mcp-build-chess/ run chess
```
